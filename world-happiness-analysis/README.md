# World Happiness Analysis

Machine Learning project that will investigate the factors affecting happiness among different nations using the data set of World Happiness Report. The project will use EDA and Multiple Linear Regression for identification of best predictor variables of happiness of nations.

## Tools

- Python
- Pandas
- Matplotlib
- Scikit-learn

## Highlights

- Data cleaning and preparation
- Exploratory data analysis (EDA)
- Correlation analysis
- Multiple Linear Regression
- Model evaluation
