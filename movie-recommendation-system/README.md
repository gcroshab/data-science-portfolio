
# Movie Recommendation System

Movie recommendation system was developed using the MovieLens dataset and item-based collaborative filtering method. The system recommends movies that are similar based on users' rating patterns using cosine similarity.

## Tools

- Python
- Pandas
- NumPy
- Scikit-learn

## Highlights

- Data preprocessing
- User-item matrix
- Cosine similarity
- Item-based collaborative filtering
- Movie recommendation engine
