# Childcare Prices Analysis and Affordability in the United States

Interactive Tableau and canva dashboard analyzing childcare prices and affordability across the United States using the National Database of Childcare Prices (NDCP).

## Tools

- Tableau
- Canva
- Data Visualization

## Highlights

- Childcare prices by state
- Regional affordability analysis
- Household income vs. childcare costs
- Interactive dashboard and visualizations
