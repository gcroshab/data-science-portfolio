# Netflix Global Viewership Analysis

Project that utilizes Python for the analysis of trends related to the Netflix Global Top 10. This project focuses on analyzing the trends in audience engagement in different countries and languages and different types of content.

## Tools
- Python
- Pandas
- Matplotlib

## Highlights
- Data preparation and cleaning
- Exploratory data analysis (EDA)
- Six data visualizations
- Global content trend analysis
