# AI Test Case Generator

This project uses Generative AI to automatically generate software test cases from user requirements.


## Technologies

- Python
- Streamlit
- OpenAI API
- Jupyter Notebook

## Features
- Generates functional, negative and edge test cases
- Produces structured test case output
- Supports prompt engineering for improved test quality

## Repository Contents
- Source code
- Streamlit application
- Final report
- Presentation

## Future Improvements
- Fine-tune the model using QA datasets
- Export test cases to Excel or JSON
- Support API and UI test case generation
