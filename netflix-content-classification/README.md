# Netflix Content Classification

Machine learning project that finds out whether a Netflix title is a Movie or TV Show using metadata features. The project applies data preprocessing, feature engineering, and logistic regression to automate content classification.

## Tools

- Python
- Pandas
- Scikit-learn
- Matplotlib

## Highlights

- Data cleaning and preprocessing
- Feature engineering
- Exploratory data analysis (EDA)
- Logistic Regression
- Model evaluation
